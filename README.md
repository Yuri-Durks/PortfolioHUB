# PortfolioHUB

## Sobre o Projeto
Este repositório foi desenvolvido com o objetivo de armazenar projetos acadêmicos e pessoais utilizando Git e GitHub.

O projeto foi criado para praticar:
- Versionamento de código;
- Organização de repositórios;
- GitHub Pages;
- Documentação técnica;
- Compartilhamento profissional.
